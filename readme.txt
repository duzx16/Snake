程序是一个双人对战的贪吃蛇小游戏。运行时由一方作为服务器，另一方作为客户端连接到服务器上。具体以服务器模式还是客户端模式运行可以在游戏开始时选择。
服务器一方只需要输入监听的端口，同时在IP地址栏里会显示服务器的IP（因为实际网络情况复杂，比如可能同时有内网IP和外网IP，所以这里给出的IP不一定准确），点击“Play”进行监听。客户端一方输入服务器的IP和端口，点击“Play”进行连接。在监听和等待连接的过程中随时可以再次点击“Play”停止。在主界面上也会有相应的提示。连接成功后程序会自动进入游戏界面。
在游戏界面按照要求服务器端（玩家1）用WASD控制，客户端（玩家2）用上下左右控制。如果发现操作无反应可能是此时焦点不在游戏主界面上（比如焦点在滑动条上或者聊天界面上），点击游戏主界面即可解决。
界面上方是双方的得分和剩余生命统计。下方是控制游戏的工具栏。滑动条可以调节蛇的移动速度。右下方的按钮分别是暂停、继续、返回主界面、音乐播放。右侧是聊天界面。
当有一方的蛇入洞时屏幕上会出现提示。当游戏暂停时游戏界面会变暗并出现提示。当一方的蛇全部死亡之后，游戏结束，弹出对话框显示游戏结果和龙虎榜。
实现的扩展要求包括：
1.背景音乐播放（自带了背景音乐，也可以自己选择音乐文件来播放）
2.龙虎榜功能（游戏结束时在对话框中显示，数据存储在游戏运行目录下的record.txt）
3.聊天功能（位于游戏界面右方）
4.动画效果（吃到食物时的动画和蛇死亡时渐隐的动画）
